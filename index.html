2 .introducción

Distribución de Frecuencias – Desarrollo Teórico

1. Concepto

La distribución de frecuencias es una técnica estadística que permite organizar, resumir y presentar datos de manera estructurada, mostrando cuántas veces aparece cada valor o categoría dentro de un conjunto de datos.

Formalmente, dado un conjunto de datos X = {x_1, x_2, \dots, x_n}, una distribución de frecuencias asigna a cada valor o clase x_i un número f_i que representa su frecuencia.

2. Tipos de Distribución de Frecuencias

2.1. Distribución de frecuencias no agrupadas

Se utiliza cuando el número de datos es pequeño y cada valor se presenta individualmente.

X = {x_1, x_2, \dots, x_n}

Cada valor tiene asociada una frecuencia absoluta f_i.

2.2. Distribución de frecuencias agrupadas

Se utiliza cuando los datos son numerosos. Los valores se agrupan en intervalos o clases.

Cada clase tiene la forma:

[L_i, U_i)

donde:

( L_i ): límite inferior ( U_i ): límite superior

3. Tipos de Frecuencias

3.1. Frecuencia absoluta ( f_i )

Número de veces que aparece un valor o clase.

f_i = \text{número de observaciones en la clase } i

3.2. Frecuencia absoluta acumulada ( F_i )

Suma de las frecuencias absolutas hasta la clase ( i ).

F_i = \sum_{j=1}^{i} f_j

3.3. Frecuencia relativa ( h_i )

Proporción de datos en cada clase respecto al total.

h_i = \frac{f_i}{n}

donde: n = \sum_{i=1}^{k} f_i

3.4. Frecuencia relativa acumulada ( H_i )

Suma acumulada de las frecuencias relativas.

H_i = \sum_{j=1}^{i} h_j

3.5. Frecuencia porcentual ( p_i )

Expresa la frecuencia relativa en porcentaje.

p_i = h_i \times 100

3.6. Marca de clase ( m_i )

Valor representativo de cada intervalo.

m_i = \frac{L_i + U_i}{2}

4. Propiedades de la Distribución de Frecuencias

4.1. Suma de frecuencias absolutas

\sum_{i=1}^{k} f_i = n

4.2. Suma de frecuencias relativas

\sum_{i=1}^{k} h_i = 1

4.3. Suma de frecuencias porcentuales

\sum_{i=1}^{k} p_i = 100

4.4. Monotonía de la frecuencia acumulada

F_1 \leq F_2 \leq \dots \leq F_k = n

y

H_1 \leq H_2 \leq \dots \leq H_k = 1

4.5. Relación entre frecuencia absoluta y relativa

h_i = \frac{f_i}{n} \quad \Longleftrightarrow \quad f_i = n \cdot h_i

5. Número de clases (Regla de Sturges)

Para datos agrupados, el número de clases ( k ) se puede estimar como:

k = 1 + 3.322 \log_{10}(n)

6. Amplitud de clase

La amplitud o ancho de clase ( A ) se calcula como:

A = \frac{R}{k}

donde el rango ( R ) es:

R = x_{\max} - x_{\min}

7. Densidad de frecuencia (para clases desiguales)

d_i = \frac{f_i}{A_i}

donde ( A_i ) es la amplitud de la clase ( i ).

8. Estructura de una Tabla de Distribución de Frecuencias

Una tabla típica contiene las siguientes columnas:

Clase Marca de clase Frecuencia absoluta Frecuencia acumulada Frecuencia relativa Frecuencia relativa acumulada
C_i x_i f_i F_i h_i H_i


Tablas de doble entrada

Estructura de una Tabla de Doble Entrada

Una tabla de doble entrada (o tabla de contingencia) es una herramienta que permite organizar y resumir datos correspondientes a dos variables simultáneamente, mostrando cómo se distribuyen de manera conjunta.

Componentes principales

Una tabla de doble entrada está compuesta por:

  1. Filas Representan las categorías de una primera variable (Variable A).

  2. Columnas Representan las categorías de una segunda variable (Variable B).

  3. Celdas internas Contienen las frecuencias (absolutas o relativas) que corresponden a la combinación de ambas variables.

  4. Totales marginales

    • Totales por fila (suma horizontal)
    • Totales por columna (suma vertical)

  5. Total general Suma total de todas las observaciones (n)

Estructura general

Categoría B1 Categoría B2 … Total
Categoría A1 n₁₁ n₁₂ … n₁•
Categoría A2 n₂₁ n₂₂ … n₂•
… … … … …
Total n•₁ n•₂ … n


Notación

  • n_{ij}: frecuencia en la fila i y columna j
  • n_{i\cdot}: total de la fila i
  • n_{\cdot j}: total de la columna j
  • n : total general

Tipos de frecuencias

Dentro de la tabla se pueden incluir:

  • Frecuencias absolutas: número de observaciones
  • Frecuencias relativas: proporciones respecto al total
  • Frecuencias porcentuales: proporciones en porcentaje
